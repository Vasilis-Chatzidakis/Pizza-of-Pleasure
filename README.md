# Pizza of Pleasure

This is an online pizza shop application for a pizza restaurant created with HTML, CSS/TailwindCSS, React(Router,Redux) and it's made to be responsive with TailwindCSS. The capabilities of this application are:

1. On the homepage when the user submits his/her full name and a button pops up. By pressing it, it loads the menu of the restaurant.
2. By adding an item to the cart the user has the following options, to increase or decrease the number of the item that you want and you can also remove it from the cart.
3. By opening the cart, it takes the user to the cart overview page and he/she has the following actions that can make: A) increase or decrease the item as well as removing it, B) go back to the menu list, C) clear the cart, D) proceed to order.
4. By procceding to order, it takes the user to a form that the he/she has to fill in order to proceed and finalize his/her order (for the location instead of typing the address, the user can use the 'GET POSITION' button, by pressing it a geolocation API will fill the users address) and the user has the option to prioritize his/her order at an extra cost.
5. After the user place the order, it takes the user at the order overview page which there the user can see the usefull informations about his/her order and have the option to prioritize the order at an extra cost.
6. By coping the order id name without the '#' special character, he/she can search the order on the search bar and view the order informations as well as the status of the order.
